# Book Management  
도서 관리 데스크톱 애플리케이션 (C# WinForms)

---

## Overview
Book Management는 도서 정보를 효율적으로 관리하기 위해 개발한  
**Windows 기반 도서 관리 프로그램**입니다.

사용자는 도서 등록, 조회, 수정, 삭제 등 기본적인 관리 기능을  
GUI 환경에서 직관적으로 수행할 수 있으며,  
WinForms를 활용해 **이벤트 기반 화면 처리와 데이터 관리 흐름**을 구현했습니다.

본 프로젝트는 C# 데스크톱 애플리케이션 구조와  
폼(Form) 간 화면 전환, 데이터 처리 로직 이해를 목표로 제작되었습니다.

---

## Key Features

### 📚 도서 관리 기능
- 도서 정보 등록
- 도서 목록 조회
- 도서 정보 수정
- 도서 정보 삭제

### 🖥 UI / 화면 구성
- Windows Forms 기반 GUI
- 버튼 이벤트를 통한 기능 처리
- 폼(Form) 간 화면 전환
- 사용자 입력값 검증 처리
